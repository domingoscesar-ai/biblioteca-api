# 👋 Biblioteca-api 
It is a project to show the process to create API using Laravel Framework. It's only a demostrative project to ensure all ability for developing server-side software and allow any client-side software comunincate and takes all resource of it. 

## 🚀 Stacks
- PHP;
- Laravel;


## List Tasks:
- Database Setup (Models, migrations);
- Creating controllers and perform all the necessary logics;
- Adding Request Classes for validations;
- Creating Resources for better data transformation & presentation;

## Endpoints:
**Authors:**
- GET /api/authors -> Lista all authors;
- POST /api/authors -> Create new author;
- GET /api/authors/{id} -> Get specific author;
-  PUT /api/authors/{id} -> Update author;
- DELETE /api/authors/{id} -> Delete author; 

**Books:**
- GET /api/books -> Lista all books;
- POST /api/books -> Create new book;
- GET /api/books/{id} -> Get specific books;
-  PUT /api/books/{id} -> Update books;
- DELETE /api/books/{id} -> Delete books;

**Members:**
- GET /api/members -> Lista all members;
- POST /api/members -> Create new member;
- GET /api/members/{id} -> Get specific member;
-  PUT /api/members/{id} -> Update member;
- DELETE /api/members/{id} -> Delete member;
